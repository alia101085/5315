1.      Summarize D3JS (https://d3js.org/) and C3JS (http://c3js.org/)

JS is java script language for client side coding language like HTML, CSS, JS+Image Processing. Compared to Sever-side which is PHP, Ruby. However now there is a java script for server side called Node to unify the two side. D3js and C3js are both java script language. D3 (Data-Driven Documents or D3.js) is a JavaScript library for visualizing data using web standards. D3 helps you bring data to life using SVG, Canvas and HTML. D3 combines powerful visualization and interaction techniques with a data-driven approach giving full capabilities of modern browsers and the freedom to design the right visual interface. While C3 is a D3-based reusable chart library that enables deeper integration of charts into web applications. Because of the dependence on D3, C3 supports only modern browsers D3 supports. C3 (Comfortable, Customizable, Controllable or C3.js) enables deeper integration of charts into our application. C3 makes it easy to generate D-3 based charts by wrapping the code required to construct the entire chart and make it customizable by enabling custom style by class and the capability of callbacks APT to update chart even after it’s rendered.
