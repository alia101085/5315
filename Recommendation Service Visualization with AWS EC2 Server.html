1.      Summarize Python Flask

Flask is a small and powerful web framework for Python. It's easy to learn and simple to use, enabling us to build our web app in a short amount of time. For example, app.py is a Flask web application that defines a RESTful-like API around the engine. In our tutorial we built a RESTful API around a Spark recommendation engine using the Flask micro-framework

2.      Describe REST (Representational State Transfer)

REST (Representational State Transfer) considered to be a set of principles that assign how HTTP and URLs Web standards are used. The main idea is that if we use REST principles while developing our application we will have a system that uses the Webs architecture to your benefit. Basically, its an architectural style and approach to communications often used in web services development. Its application program interface (API) uses HTTP requests to GET, PUT, POST and DELETE data.
